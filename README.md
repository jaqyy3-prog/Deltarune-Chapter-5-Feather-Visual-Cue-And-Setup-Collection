# Deltarune Chapter 5 Feather Visual Cue and Setup Collection
This is a compilation of visual cues or setups that I personally use throughout CH5 runs!! Not all of these are necessary to be optimal, but I find that many of them make optimization easier. I have chosen to include mostly non-obvious cues and setups, so this is not a comprehensive list of all strats in the run.
In general, the cues are used to greed movement, usually landing on a platform as soon as possible or getting off a moving platform as soon as possible.
